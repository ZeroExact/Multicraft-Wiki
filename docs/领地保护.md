# 领地保护

你可以在生存服创建领地, 以保护一片区域，初始最多可以创建 1 个

> 打开主菜单找到 `权限升级` 一项，可花费CC增加数量

领地价格 0.384 / 格 （忽略Y轴）

大小 20×20 起圈，这个是为了尽可能避免有人进服圈了个小地就再也不上线

## 腐竹怎么圈地？

掏出你的 `木锄` ，没有就去做一把，没材料就公屏打字找个人借一把

> 根据我们小学二年级学过的平面几何基础，两点可以确定一个长方形区域

左键选择第一个点，右键选择第二个点，高度任意，因为强制帮你从-64圈到384

选完两个点你应该会看到你已选择区域大小和价格的提示

输入 `/res create <领地名>` 以创建你的领地

为什么强制圈满高度？上周目总有人自己不圈满然后被幻翼追着杀来问我啥时候关幻翼

简单，我帮你圈满就行了

## 权限设置

输入 `/res set` 可以打开设置GUI，这里的设置是全局的

![](_images/领地保护/权限GUI.jpg)

如果要对个人进行设置，输入 `/res pset <玩家ID> <权限节点> <t/f>`

例如关闭 theFSO 的 use 权限：`/res pset theFSO use f`

以下是常用指令：

* `/res set` - 打开领地设置面板'
* `/res set <权限标识> <t/f>` - 设置领地全局权限'
* `/res pset <玩家名> <权限标识> <t/f>` - 设置某个玩家的权限'
* `/res padd <玩家名>` - 给予玩家常用权限'
* `/res info` - 查看当前位置的领地信息'
* `/res select size` - 查询选区信息'
* `/res remove <领地名>` - 删除领地'
* `/res give <领地名> <玩家名>` - 将领地给予玩家'
* `/res set tp t` - 设置允许TP到、TP进领地'
* `/res tp <领地名>` - 传送到领地'
* `/res tpset` - 设置领地TP点的位置'

更多指令用法在游戏内输入 `/res help` 查看
