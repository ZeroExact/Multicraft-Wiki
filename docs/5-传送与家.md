# 传送与家

你可以请求传送到其他玩家或者是让对方传过来

指令分别是 `/tpa <玩家ID>` 和 `/tpahere <玩家ID>`

服务器设有一些地标点也就是 warp 点，输入 `/warp <地标名称>` 就可以传送过去

可以输入 `/sethome [家名称]` 来设置一个家，初始最多可以设置 1 个

> 打开主菜单找到 `权限升级` 一项，可花费CC增加数量

然后你可以通过 `/home [家名称]` 指令返回

如果你不输入家名称，那太好了你直接 `/home` 就回去了

## 常用指令

这么基础的事情我觉得大家应该都懂，不懂的玩玩也懂

所以简单把常用指令贴出来就不细讲了

* `/tpa <玩家ID>` - 请求传送到其他玩家'
* `/tpahere <玩家ID>` - 请求其他玩家传送到你身边'
* `/tpaccept` - 同意传送申请'
* `/tpdeny` - 拒绝传送申请'
* `/tpignore` - 启用或禁用屏蔽传送请求'
* `/warp <地标名称>` - 传送到地标'
* `/sethome` - 设置默认家'
* `/sethome <家名称>` - 设置对应名字的家'
* `/delhome home` - 删除默认家'
* `/delhome <家名称>` - 删除对应名字的家'
* `/home` - 传送到默认家'
* `/home <家名称>` - 传送到对应名字的家'
* `/homelist` - 列出所有家'
