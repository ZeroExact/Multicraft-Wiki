# 服务器介绍

我们是国内资历最为悠久的服务器之一，现今已有十余年历史，曾作为MCBBS论坛官方服务器而被众多玩家熟知。我们致力于在原版的基础上加入许多奇奇怪怪、特色鲜明的独特内容，为玩家打造一个新奇有趣的游戏世界，不再如同原版一般枯燥无味。

本周目为第八周目，服务端核心版本为 Minecraft 1.20.1

我们保留了以往周目的许多传统艺能，例如收集材料完成成就获得称号、特殊材料兑换等等，并在此基础上加入了原创的特色新内容，例如效果各异且词条随机的武器装备、鉴定人品的武器强化与镶嵌系统等等···

* 账号使用

微软要求仅允许为付费玩家提供服务器游玩服务，我们在本周目启用正版验证，强制要求使用微软账号进入服务器。Multicraft 账号已弃用，但你依然可以使用我们的外置登录库与皮肤站功能。
Multicraft 7 中的特定数据可以被继承至本周目，请在服务器内输入 `/menu` 打开主菜单，找到 `账号继承` 一项查看。
