# 介绍

说是RPG装备，其实没有特别RPG，只是大家都喜欢叫RPG装备，所以姑且就这么叫吧

在你亲手击杀绝大部分敌对怪物时，有概率掉落一类随机装备，他们的名字、材质、品质、词条等等属性都是随机的

你还可以通过一些特殊的方式获得限定装备，他们通常拥有专属名字，无限耐久，词条固定不随机

这些装备拥有一套特殊的属性机制，你可以使用锻造石升级至最高90级，提高面板数值，可以镶嵌战灰赋予战技，或是镶嵌珠宝提供加成等等

为什么说没有特别RPG呢，首先我们不是RPG服，没有等级、职业、刷副本这些内容玩法

RPG装备提供的属性也就是额外加入了一些例如暴击伤害、伤害减免、防御力等等很常见的属性

而没有什么元素伤害啊吸血啊之类的，珠宝镶嵌也只是提供这些词条的数值或者额外提供原版有的BUFF效果和附魔
