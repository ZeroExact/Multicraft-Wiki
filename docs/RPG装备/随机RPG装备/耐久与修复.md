# 耐久与修复

拥有和原版一样的耐久上限与消耗机制，可以使用原版的方式通过铁砧修复，也可以使用修复石来修复

> 经验修补附魔无法对此类物品生效

根据品质的不同，耐久类型也不同

* 一星：耐久耗尽后将会损毁消失
* 二星：耐久耗尽后将会损毁消失
* 三星：耐久耗尽后物品不损毁
* 四星：耐久耗尽后物品不损毁
* 五星：耐久耗尽后物品不损毁
* 限定：无限耐久

五星物品有小概率无限耐久，而限定则必定是无限耐久  
如何判断无限耐久呢，很简单，如果物品没有显示耐久，那就是无限

三星及以上的物品，耐久耗尽时，将无法使用、无法提供加成
