# 品质

一共有5种基本类型的品质与多种特殊类型的品质：

* 一星：普通
* 二星：稀有
* 三星：精湛
* 四星：奇幻
* 五星：史诗
* 特殊：限定、幽邃、违禁

其中限定指的是特殊奖励、星券购买获得的物品，一般是无限耐久

幽邃为幽邃锻造石专有品质

违禁顾名思义，无法使用

## 品质影响什么？

* 锻造升级

  品质越高，锻造加成系数越高，同时物品自身锻造成功率越低，越难锻造


* 自带附魔

  品质越高，自带的随机附魔上限越屌


* 镶嵌槽位数量

  品质越高，镶嵌槽位越多，简单易懂，但无论品质如何，都会至少有一个战灰槽与一个珠宝槽


* 耐久类型

  一星与二星在耐久耗尽时物品将会损毁消失，而三星、四星、五星则不会消失，而是需要修复后才能继续使用

  五星有概率出现无限耐久，限定五星则固定无限耐久  


* 分解产物

  品质越高，分解产物越好越多
