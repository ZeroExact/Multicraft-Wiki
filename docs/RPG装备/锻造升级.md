# 锻造升级

可以使用 `锻造水晶` 对物品进行锻造升级，绝大多数物品等级上限为90级

![](_images/锻造升级/锻造水晶.png)

> 锻造水晶通过挖掘紫水晶母岩概率掉落

# 锻造加成

升级时，将会提升物品特定的属性，例如攻击力、暴击伤害、防御力等等

> 最终锻造成功率 = 物品锻造成功率 × 锻造水晶锻造力

已镶嵌的珠宝、战灰提供的数值不会被强化，而是独立加法计算

# 物品锻造成功率

品质越高，物品自身锻造成功率越低

* 一星：90%
* 二星：80%
* 三星：70%
* 四星：60%
* 五星：50%

# 锻造水晶锻造力

品质越高，锻造水晶锻造力越高

* 初等：60%
* 优质：90%
* 精炼：120%
* 完美：150%
* 幽匿：180%
