# 违规与作弊

作为商业服，我们有义务尽最大努力保障所有玩家的利益，包括但不限于平衡性方面，因此我们严禁任何影响平衡的违规与作弊

所有作弊行为均由反作弊系统自行检测并记录、通报管理人员，最后由管理人员人工授权封禁并记录

管理人员有权力无视反作弊系统报告，自行判断复杂情形并人工封禁玩家

> 告诫全体玩家，请勿使用任何作弊客户端或搭载了作弊类MOD的客户端进入服务器，只要使用，就有封禁的可能性

## 违规行为判定

我们认定的违规行为包括但不限于以下情形：

* 出于恶意地使用恶性漏洞批量刷取、复制物品或获得服务器货币（刷线等原版特性暂不在此列）
* 出于恶意地使用恶性漏洞致使服务器卡顿、崩溃
* 在游戏公屏、QQ交流群（不包含撕逼群）内恶意攻击他人
* 批量使用异常账号（例如黑卡、被盗账号）进入服务器

## 作弊行为判定

我们认定的作弊包括但不限于使用作弊客户端、作弊MOD等手段实现以下功能：

* 作弊飞行/滞空
* 提高角色移动速度
* 提高角色跳跃高度
* 提高角色破坏方块的速度或其取消间隔
* 自动索敌并攻击（杀戮光环）
* 自动有目的性地追踪并破坏方块

暂不将以下功能列入判定范围：

* 自由视角
* 自动钓鱼
* 永久夜视
* 外部脚本实现的操作（鼠标连点，持续前进等）
* 投影显示与自动建造

## 连坐判定

一旦某个账号被封禁，将链式连坐封禁涉案账号IP登录过的所有账号，因此不要将账号交给你不熟的人
