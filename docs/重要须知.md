# 重要须知

进来玩之前，你应该至少先读明白下面这些东西

在服里游玩就代表你无条件遵守以下有关守则

## 账号

微软要求第三方服务器仅允许为付费玩家提供服务器游玩服务，所以我们在这周目启用正版验证，也就是必须要用微软账号登录服务器。此前使用的 Multicraft 账号已弃用，但你依然可以使用我们的外置登录库与皮肤站功能。

Multicraft 7 中的特定数据可以被继承至本周目，请在服务器内输入 `/menu` 打开主菜单，找到 `账号继承` 一项查看。

## 特性与机制

我们不是“生电服”，为保证游戏性，修复了原版一些会影响到平衡的特性，例如刷沙、刷铁轨等等。我们还根据游戏内容设计策划，额外对部分原版特性做出了修改，禁止或限制了一些行为操作。但请放心，我们会最大程度地保留原版的玩法乐趣。

## 反作弊

严禁任何形式的作弊，包括但不限于杀戮光环、违规飞行等作弊功能。所有玩家一视同仁，管理组将在查证后永久封禁相关违规账号并封存其服务器内资产。

## 氪金充值

严禁未满18岁的中华人民共和国公民充值，若有所隐瞒并引发退款纠纷，查证后将全额退款，永久封禁其账号并封存其服务器内资产。

## 人民币交易

管理组不建议、不介入玩家之间任何形式的现实货币交易，不处理其产生的纠纷，一切风险与后果均由当事人自行承担。
