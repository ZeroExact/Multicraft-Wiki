# 生物限制

每个区块可以同时存在的生物存在上限

当区块内生物数量超过上限时，后被计算的生物将会被优先清理消失

以下是详细列表与限制数量，合在一起写就是共同计算

## 友好生物

> 每 5 分钟清理一次

* 牛、哞菇、鸡、绵羊、猪、兔子：8
* 猫、狗/狼、鹦鹉：8
* 美西螈、狐狸：8
* 马、骷髅马、僵尸马: 8
* 驴、骡、羊驼/行商羊驼：8
* 蝌蚪、青蛙、海龟：8
* 铁傀儡、雪傀儡：2
* 熊猫：4
* 蜜蜂：8
* 村民：4
* 山羊：4
* 嗅探兽：4
